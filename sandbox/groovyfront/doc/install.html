Install the GroovyFront into an Ant installation

Actually you just need to make the groovyfront.jar and its dependences available in the classpath of Ant when it starts. It has to be added the same way the Ant's optional dependencies have to be added: see in the Ant manual how to install an optional task.

The jars to be installed in Ant:

  • GroovyFront which you should have build: ant-groovyfront.jar
  • Groovy 1.7.0: groovy-1.7.0.jar
  • ASM 3.2: asm-3.2.jar
  • Antlr 2.7.7: antlr-2.7.7.jar
