GIT Repositories

Access the Source Tree (GIT)

Anyone can checkout source code from our public GIT repository. To do so, simply use the following command (if you are using a GUI client, configure it appropriately):

git clone https://git-wip-us.apache.org/repos/asf/ant-[project]

Modules available for access are:

  • ant - The "main" Ant module.
    https://git-wip-us.apache.org/repos/asf/ant.git
  • Ivy - The "main" Ivy module.
    https://git-wip-us.apache.org/repos/asf/ant-ivy.git
  • IvyDE - Eclipse Plugin for Ivy.
    https://git-wip-us.apache.org/repos/asf/ant-ivyde.git
  • antunit Test framework for Ant.
    https://git-wip-us.apache.org/repos/asf/ant-antlibs-antunit.git
  • Common scripts/docs for antlibs Common files used to maintain the Antlibs
    https://git-wip-us.apache.org/repos/asf/ant-antlibs-common.git
  • Compress Antlib Antlib containing archiving and compression related tasks and types based on Apache Commons Compress
    https://git-wip-us.apache.org/repos/asf/ant-antlibs-compress.git
  • Dotnet antlib Contains Nant, Nunit, ...
    https://git-wip-us.apache.org/repos/asf/ant-antlibs-dotnet.git
  • Props antlib Properties Antlib, ...
    https://git-wip-us.apache.org/repos/asf/ant-antlibs-props.git
  • svn antlib Contains tasks to interact with Subversion repositories.
    https://git-wip-us.apache.org/repos/asf/ant-antlibs-svn.git
  • VSS antlib Visual Sourcesafe Antlib, ...
    https://git-wip-us.apache.org/repos/asf/ant-antlibs-vss.git
  • easysant buildtypes EasyAnt buildtypes
    https://git-wip-us.apache.org/repos/asf/ant-easyant-buildtypes.git
  • easysant core EasyAnt core
    https://git-wip-us.apache.org/repos/asf/ant-easyant-core.git
  • easysant4e EasyAnt4e
    https://git-wip-us.apache.org/repos/asf/ant-easyant-easyant4e.git
  • easysant plugins EasyAnt plugins
    https://git-wip-us.apache.org/repos/asf/ant-easyant-plugins.git
  • easysant buildtypes EasyAnt buildtypes
    https://git-wip-us.apache.org/repos/asf/ant-easyant-skeletons.git
  • easysant tasks EasyAnt tasks
    https://git-wip-us.apache.org/repos/asf/ant-easyant-tasks.git

If you are not familiar with Git, Apache Software Foundation's Git source repositories page may hold many helpful hints.
