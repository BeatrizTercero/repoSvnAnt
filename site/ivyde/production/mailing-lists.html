Mailing Lists


Before posting to mailing lists, make sure you don't have the answer to your question in the FAQ or more generally in the documentation.

Actually there is no Apache IvyDE specific mailing lists. Every discussion about Apache IvyDE is part of the Apache Ivy mailing lists. So if you have any question:
  • about the use of Apache IvyDE, use ivy-user@ant.apache.org.
  • about the development use dev@ant.apache.org.
  • and to get the subversion and Jira notifications, subscribe to notifications@ant.apache.org.
For further information about these mailing lists, see Ivy mailing list page.
